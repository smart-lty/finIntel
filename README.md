# 代码运行

按照以下指示运行即可：

1. 创建虚拟环境

   `conda create -n text_cnn python=3.5`

2. 激活虚拟环境

   `conda activate text_cnn`

3. 安装相关依赖

   `pip install -r requirements.txt`

4. 运行

   `python user.py`
